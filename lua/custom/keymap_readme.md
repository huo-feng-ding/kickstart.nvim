## 附加信息

### 系统剪贴板配置

已配置系统剪贴板同步（`+` 寄存器）：
- Windows: 使用 `win32yank.exe` 或 `clip.exe`
- WSL: 使用 `win32yank.exe -i --crlf`

### 文件编码

默认文件编码：UTF-8  
支持编码：UTF-8, GB18030, GBK, GB2312, CP936, USC-BOM, EUC-JP

### 自动命令

- `TextYankPost`: 复制时高亮文本
- `BufReadPost`: 打开文件时恢复上次光标位置
- `FileType`: 新建行时不自动添加注释符号
- `LspAttach`: LSP 附加时启用功能
- `PackChanged`: 插件安装/更新时运行构建

## 快捷键前缀说明

当前配置使用 `<Space>` 作为 Leader 键：

```
<Space> + [key] = [功能]
```

例如：
- `<Space> + c` + `f` = 复制文件名 (`<leader>cf`)
- `<Space> + c` + `c` = 复制文件路径 (`<leader>cc`)
- `<Space> + s` + `f` = 查找文件 (`<leader>sf`)
